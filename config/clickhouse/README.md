# ClickHouse Configuration

ClickHouse configuration files for TelemetryFlow Core audit logging.

## Files

- `config.xml` - Main ClickHouse server configuration
- `users.xml` - User accounts and access control

## Usage

These configuration files are used when deploying ClickHouse for audit log storage.
